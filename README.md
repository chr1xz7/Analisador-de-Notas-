# Programação-em-C
Repositório dedicado ao estudo da linguagem C: algoritmos, estruturas de dados e exercícios práticos.
Este repositório foi criado para organizar e expor meus códigos, exercícios e projetos desenvolvidos enquanto estudo a linguagem C.
O objetivo principal é praticar a lógica de programação, o gerenciamento de memória e os conceitosfundamentais da linguagem.
